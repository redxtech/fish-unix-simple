# unix-simple

> unix-simple is a fish plugin that provides a function that shows a graphic about unix

## installing

### fisher

```fish
fisher install redxtech/fish-unix-simple
```

## usage

`unix-simple` is very easy to use, simply run the command:

```fish
unix-simple
```
